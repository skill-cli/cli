# Code of Conduct

This project expects direct, respectful collaboration.

## Expected Behavior

- Keep technical criticism specific and actionable.
- Assume changes are made in good faith unless there is clear evidence
  otherwise.
- Make room for disagreement while staying focused on the repository's goals.
- Avoid harassment, personal attacks, and disruptive behavior.

## Scope

This code of conduct applies to repository issues, pull requests, discussions,
reviews, and project-maintained communication spaces.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or limit
participation when conduct harms the project or contributors.
